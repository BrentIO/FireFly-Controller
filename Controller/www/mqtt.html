MQTT
Configure the MQTT broker that controllers and clients will connect to. All devices will use the same configuration.

Wildcards

You can configure the username and password to use case-sensitive wildcards based on the devices' hardware:
Wildcard Example Value
$$mac$$ 
DEADBEEFFEED
$$mac_dashes$$ 
DE-AD-BE-EF-FE-ED
$$mac_colons$$ 
DE:AD:BE:EF:FE:ED
$$uuid$$ 
b113d8ff-51ef-4fd8-82c0-7dac74d73ef3
